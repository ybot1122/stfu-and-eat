STFU and EAT is a no-nonsense recipe blog that optimizes for macros and flavor. We focus on straightforward, macro-friendly recipes that maximize taste with minimal fuss.

The site includes features like [Pagefind](https://pagefind.app) for search and [Giscus](https://giscus.app) for comments. For a list of updates and changes, see this [blog post](https://astro-micro.vercel.app/blog/00-micro-changelog).

STFU and EAT also offers a sitemap, an RSS feed, and Markdown + MDX support. It's styled with TailwindCSS and supports system, light, and dark themes.
